
\chapter{Plan de Seguridad y Salud}
\label{chap:plan_seguridad}

La instalación del sistema de cableado estructurado comprende actividades de canalización, perforación, tendido de cables, terminación de puntos de red, montaje del MDF y pruebas de certificación. Estas tareas pueden generar riesgos para los técnicos, los usuarios del edificio y la infraestructura existente.

El presente plan establece las medidas preventivas necesarias para ejecutar los trabajos de manera segura, evitando accidentes, daños materiales e interrupciones innecesarias de las actividades académicas.

\section{Normas de Seguridad Eléctrica y Física}

Antes de iniciar cada jornada de trabajo, el responsable de seguridad deberá revisar las condiciones del área, las herramientas y los equipos que serán utilizados. Asimismo, se realizará una charla breve para informar las actividades previstas y los riesgos asociados.

Las principales medidas de seguridad serán las siguientes:

\begin{itemize}
    \item Delimitar y señalizar las zonas de trabajo.
    \item Mantener libres las rutas de evacuación y los pasillos.
    \item Utilizar herramientas en buen estado.
    \item Retirar diariamente restos de cable, canaletas y empaques.
    \item Evitar dejar materiales o herramientas en zonas de circulación.
    \item Coordinar los trabajos ruidosos en horarios de menor ocupación.
    \item Suspender cualquier actividad cuando exista una condición insegura.
\end{itemize}

Cuando se trabaje cerca de instalaciones eléctricas, se deberá verificar previamente la ubicación de los conductores y mantener la separación correspondiente entre el cableado de datos y el cableado de energía.

Si la actividad requiere intervenir cerca de un circuito eléctrico, se aplicará el siguiente procedimiento:

\begin{enumerate}
    \item Coordinar la interrupción del suministro eléctrico.
    \item Desenergizar el circuito correspondiente.
    \item Evitar su reactivación accidental.
    \item Verificar la ausencia de tensión.
    \item Ejecutar el trabajo.
    \item Retirar las herramientas y protecciones.
    \item Restablecer el servicio únicamente con autorización.
\end{enumerate}

No se utilizarán canalizaciones eléctricas existentes para instalar cables de telecomunicaciones.

\section{Procedimientos para Trabajos en Altura y Canalizaciones}

La instalación de puntos para Access Points, cámaras, canaletas y conduit puede requerir trabajos en zonas elevadas. Para estas actividades se deberán cumplir las siguientes medidas:

\begin{itemize}
    \item Utilizar escaleras industriales o andamios en buen estado.
    \item Colocar la escalera sobre una superficie firme y nivelada.
    \item No utilizar mesas, sillas ni mobiliario como plataforma.
    \item Mantener tres puntos de contacto durante el ascenso.
    \item No subir con herramientas sueltas en las manos.
    \item Utilizar cinturón o bolsa portaherramientas.
    \item Contar con una segunda persona de apoyo desde el piso.
    \item Delimitar la zona inferior para evitar el tránsito de personas.
    \item Utilizar un sistema anticaídas cuando la evaluación del riesgo lo requiera.
    \item Suspender el trabajo si la escalera, el piso o la superficie presentan inestabilidad.
\end{itemize}

Antes de perforar muros o techos se revisarán los planos y se verificará que no existan tuberías, conductores eléctricos u otros servicios ocultos.

Durante las perforaciones se deberá:

\begin{itemize}
    \item Utilizar lentes de seguridad y mascarilla.
    \item Revisar el estado del taladro y de su cable de alimentación.
    \item Sujetar correctamente la herramienta.
    \item Evitar el uso de ropa suelta.
    \item Proteger el mobiliario y los equipos cercanos.
    \item Recoger el polvo y los residuos al finalizar.
    \item Mantener alejadas a las personas no autorizadas.
\end{itemize}

Las canaletas y tuberías conduit deberán instalarse sin bordes cortantes y con una fijación adecuada para evitar desprendimientos o daños en el cableado.

\section{Seguridad en Áreas Ocupadas}

Debido a que las actividades académicas pueden continuar durante la instalación, se adoptarán medidas adicionales para proteger a estudiantes, docentes y personal administrativo:

\begin{itemize}
    \item Programar los trabajos por ambientes.
    \item Cerrar temporalmente el área intervenida.
    \item No dejar cables atravesando pasillos.
    \item Cubrir perforaciones y registros abiertos.
    \item Informar sobre restricciones temporales de acceso.
    \item Coordinar anticipadamente los cortes de red o energía.
    \item Ejecutar las actividades ruidosas en horarios de menor ocupación.
\end{itemize}

El responsable de seguridad verificará que las rutas de evacuación permanezcan libres durante toda la ejecución.

\section{Procedimiento ante Emergencias}

Ante un accidente o una condición peligrosa se aplicará el siguiente procedimiento:

\begin{enumerate}
    \item Suspender inmediatamente la actividad.
    \item Aislar y señalizar la zona.
    \item Desconectar la energía cuando sea necesario.
    \item Brindar primeros auxilios sin exponer a otras personas.
    \item Comunicar el incidente al supervisor.
    \item Solicitar asistencia médica cuando corresponda.
    \item Registrar el hecho y sus posibles causas.
    \item Aplicar medidas correctivas antes de reiniciar el trabajo.
\end{enumerate}

Durante la ejecución deberá mantenerse disponible un botiquín de primeros auxilios y conocerse la ubicación de los extintores y las rutas de evacuación del edificio.

\subsection{Orden, limpieza y manejo de residuos}

Los residuos generados serán retirados diariamente. Entre ellos se consideran:

\begin{itemize}
    \item Restos de cable.
    \item Recortes de canaletas y conduit.
    \item Bridas y etiquetas.
    \item Cajas y empaques.
    \item Polvo de perforación.
    \item Componentes defectuosos.
\end{itemize}

Los residuos deberán almacenarse temporalmente en recipientes o bolsas adecuadas, sin bloquear pasillos ni salidas.

\section{Registros de Seguridad}

Durante la ejecución se conservarán los siguientes registros:

\begin{itemize}
    \item Charlas de seguridad.
    \item Inspecciones de herramientas.
    \item Inspecciones de escaleras.
    \item Control de entrega y uso de EPP.
    \item Fotografías de la señalización.
    \item Reportes de incidentes.
    \item Acciones correctivas aplicadas.
    \item Acta de cierre de seguridad.
\end{itemize}

El proyecto se considerará ejecutado de manera segura cuando las actividades hayan concluido sin accidentes incapacitantes, sin daños a la infraestructura y cumpliendo las medidas preventivas establecidas.
